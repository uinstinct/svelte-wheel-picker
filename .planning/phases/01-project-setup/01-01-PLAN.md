**svelte.config.js** — per RESEARCH.md Pattern 2:
```javascript
import adapter from '@sveltejs/adapter-auto';
import { vitePreprocess } from '@sveltejs/vite-plugin-svelte';

/** @type {import('@sveltejs/kit').Config} */
const config = {
  preprocess: vitePreprocess(),
  kit: {
    adapter: adapter(),
    alias: {
      '$lib': 'src/lib',
    },
  },
};

export default config;
```

**vite.config.ts** — standard SvelteKit vite config:
```typescript
import { sveltekit } from '@sveltejs/kit/vite';
import { defineConfig } from 'vite';

export default defineConfig({
  plugins: [sveltekit()],
});
```

**vitest.config.ts** — per RESEARCH.md Pattern 3 (Vitest 4.x API, NOT the old string-based provider API):
```typescript
import { defineConfig } from 'vitest/config';
import { svelte } from '@sveltejs/vite-plugin-svelte';
import { playwright } from '@vitest/browser-playwright';

export default defineConfig({
  plugins: [svelte({ hot: !process.env.VITEST })],
  test: {
    setupFiles: ['vitest-browser-svelte'],
    browser: {
      enabled: true,
      provider: playwright(),
      instances: [{ browser: 'chromium' }],
    },
    include: ['src/**/*.{test,spec}.{js,ts}'],
  },
});
```

Run `pnpm install` to install all devDependencies declared in package.json.

After install completes, run `pnpm exec playwright install chromium` to download the Chromium binary required by Vitest browser mode (per RESEARCH.md Pitfall 3 -- the binary is NOT part of the npm package, it must be downloaded separately).

Then generate the `.svelte-kit/` directory by running `pnpm dev` briefly (per RESEARCH.md Pitfall 1 -- tsconfig.json extends `.svelte-kit/tsconfig.json` which does not exist until SvelteKit runs once). Start `pnpm dev`, wait for the server to initialize and create `.svelte-kit/`, then kill the process. A simpler alternative: `pnpm build` or `npx svelte-kit sync` also generates `.svelte-kit/`.

Preferred approach: `npx svelte-kit sync` -- this generates `.svelte-kit/` without starting a full dev server.

Commands in order:
1. `pnpm install`
2. `pnpm exec playwright install chromium`
3. `npx svelte-kit sync`
4. Verify: `ls .svelte-kit/tsconfig.json` (must exist)
