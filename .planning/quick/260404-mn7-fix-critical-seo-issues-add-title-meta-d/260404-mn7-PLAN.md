In Svelte, the JSON-LD script tag needs `{@html ...}` to avoid escaping. Use this pattern:
```svelte
<svelte:head>
  <!-- meta tags here -->
  {@html `<script type="application/ld+json">${JSON.stringify(jsonLdSchema)}</script>`}
</svelte:head>
```

Define `jsonLdSchema` as a const in the `<script>` block with the @graph structure above.

Do NOT remove or modify any existing markup or styles in the file.
