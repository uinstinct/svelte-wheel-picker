- `try/catch` fallback means local dev always works even without a releases branch
- deploy.yml `|| echo "No releases branch yet"` makes the fetch step non-fatal on first-ever deploy before any release exists
- Moved body padding from `body` to `main` so content starts below the sticky navbar (48px height) rather than being covered by it
