Convert the source image `og.png` (project root) to WebP format using `cwebp` (available at /opt/homebrew/bin/cwebp):

```bash
cwebp -q 90 og.png -o static/svelte-wheel-picker-og-1200x630.webp
```

Use quality 90 for a good balance of file size and visual fidelity. Verify the output file exists and is a reasonable size (should be under 200KB for an OG image).

If the source image is not exactly 1200x630, resize it first using `sips` before converting:
```bash
sips --resampleWidth 1200 --resampleHeight 630 og.png --out /tmp/og-resized.png
cwebp -q 90 /tmp/og-resized.png -o static/svelte-wheel-picker-og-1200x630.webp
```
